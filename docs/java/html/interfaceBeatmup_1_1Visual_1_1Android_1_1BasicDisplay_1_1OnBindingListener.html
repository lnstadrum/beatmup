void Beatmup.Visual.Android.BasicDisplay.OnBindingListener.beforeBinding ( boolean  valid)

Called before a display-to-context binding event happens.

The target context have to run system tasks to set up the display. If it is busy with other jobs (namely persistent ones), the binding will then be blocked and UI will be freezed.
