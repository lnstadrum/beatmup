Detailed Description

A binary operation on images.

Evaluates expression O = op(L, R) for bitmaps L, R, O and a given pixelwise operation op. Allows to select the operating area in all the three bitmaps.

Specifies a rectangular operating area.

Only pixels within a specific rectangle in left and right operands are used. The result is put in a rectangular area of the same size in the output bitmap (the remaining pixels are unchanged).
