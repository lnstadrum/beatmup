24  * Neural net model.
25  * Contains a list of operations and programmatically defined interconnections between them using addConnection().
26  * Enables access to the model memory at any point in the model through addOutput() and getModelData().
27  * The memory needed to store internal data during the inference is allocated automatically; storages are reused when possible.
28  * The inference of a Model is performed by InferenceTask.
29  */
